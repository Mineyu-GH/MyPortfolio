# RubyBookKeeper

**RubyBookKeeper** は、書籍情報を簡単に登録・管理できる蔵書管理Webアプリケーションです。  
Ruby（WEBrick, ERB, SQLite3）で構築されており、書籍のタイトル・著者・ページ数・発刊日をデータベースに保存し、Webブラウザから一覧表示・追加・編集・削除ができます。

## 制作の経緯

Webアプリケーションの仕組みについて学習しながら、簡単なアプリを一つ作ってみようと思ったことがきっかけです。今回はフロントエンドのデザインを「BootStrap」で簡単に実装できました。バックエンドの言語には簡単にコーディングできるRubyのWEBrickというフレームワークを使用しました。

## 苦労した点

HTMLのフォーム上での入力をRubyで読み込んで処理する際、何度も「ASCIIとUTF-8を変換できない」というエラーが発生し続け、それについて調べてデバッグするのに多くの時間を費やしました。

## 用いた技術
### フロントエンド

- HTML5/CSS
- ERB(テキストファイル内にコードを埋め込めるRubyライブラリ)
- BootStrap(Webアプリのフロントエンド用フレームワーク)

### バックエンド
- Ruby WEBrick(HTTPサーバーフレームワークを作れるRuby標準ライブラリ)

### データベース
- SQLite3

## 主な機能

- 書籍情報の登録（タイトル、著者、ページ数、発刊日）
- 書籍一覧の表示
- 書籍情報の編集・削除
- Bootstrapによるシンプルで見やすいUI

## 使い方

1. 必要なRubyライブラリをインストール  
   ```
   gem install sqlite3 webrick
   ```
2. サーバーを起動  
   ```
   ruby webBookManager.rb
   ```
3. ブラウザで [http://localhost:8080](http://localhost:8080) にアクセス

## ファイル構成

- `webBookManager.rb` ... メインアプリケーション
- `list.erb` ... 書籍一覧・登録画面
- `edit.erb` ... 書籍編集画面
- `book_info.sqlite3` ... 書籍データベース
